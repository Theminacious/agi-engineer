# AGI Engineer V2 Frontend

Next.js 15 dashboard for the AGI Engineer GitHub App.

## Features

- **Dashboard**: View analysis results and project health
- **Settings**: Configure per-repository analysis rules
- **History**: Track analysis runs and improvements
- **OAuth**: GitHub App one-click installation

## Development

```bash
npm install
npm run dev
```

Dashboard runs on `http://localhost:3000`

## Building

```bash
npm run build
npm start
```

## Tech Stack

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **FastAPI Backend** - API integration
